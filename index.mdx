Json Everything is a suite of .NET libraries built on top of `System.Text.Json`, providing complete implementations of major JSON standards. Each library is independently installable via NuGet, so you only pull in what you need.

## Standards compliance

Every library in this suite targets full compliance with its associated specification. JSON Schema support is independently tested against the [JSON Schema Test Suite](https://github.com/json-schema-org/JSON-Schema-Test-Suite) and results are publicly visible on [Bowtie](https://bowtie-json-schema.github.io/bowtie).
