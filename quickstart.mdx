---
title: "Quickstart: validate, query, and patch JSON in .NET"
description: "Learn how to validate JSON, query with paths, apply patches, and navigate with pointers using the json-everything library suite in four steps."
---

Json Everything is a collection of independent NuGet packages. This quickstart walks you through the four most-used libraries so you can go from installation to working code in minutes.

<Steps>
  <Step title="Validate JSON with JsonSchema.Net">
    Install the package:

    ```bash
    dotnet add package JsonSchema.Net
    ```

    Parse a schema and validate an instance:

    ```csharp
    using Json.Schema;
    using System.Text.Json.Nodes;

    // Build a schema with the fluent API
    var schema = new JsonSchemaBuilder()
        .Title("My Object")
        .Type(SchemaValueType.Object)
        .Properties(
            ("foo", new JsonSchemaBuilder().Type(SchemaValueType.String)),
            ("bar", new JsonSchemaBuilder().Type(SchemaValueType.Number))
        )
        .Required("foo")
        .Build();

    var instance = JsonNode.Parse("{\"foo\":\"hello\",\"bar\":42}");
    var results = schema.Evaluate(instance);

    Console.WriteLine(results.IsValid); // true
    ```

    <Tip>You can also load schemas from a file with `JsonSchema.FromFile("schema.json")` or from a string with `JsonSchema.FromText(json)`.</Tip>
  </Step>

  <Step title="Query JSON with JsonPath.Net">
    Install the package:

    ```bash
    dotnet add package JsonPath.Net
    ```

    Parse a path expression and evaluate it against a JSON document:

    ```csharp
    using Json.Path;
    using System.Text.Json.Nodes;

    var path = JsonPath.Parse("$.prop[0:6:2]");
    var instance = JsonNode.Parse("{\"prop\":[0,1,2,3,4,5]}");

    var results = path.Evaluate(instance);

    foreach (var match in results.Matches)
    {
        Console.WriteLine($"{match.Location}: {match.Value}");
    }
    // $['prop'][0]: 0
    // $['prop'][2]: 2
    // $['prop'][4]: 4
    ```
  </Step>

  <Step title="Navigate with JsonPointer.Net">
    Install the package:

    ```bash
    dotnet add package JsonPointer.Net
    ```

    Build or parse a pointer and extract a value:

    ```csharp
    using Json.Pointer;
    using System.Text.Json.Nodes;

    var pointer = JsonPointer.Parse("/objects/and/3/arrays");

    var node = JsonNode.Parse("{\"objects\":{\"and\":[\"item zero\",null,2,{\"arrays\":\"found me\"}]}}");
    var success = pointer.TryEvaluate(node, out var result);

    Console.WriteLine(result); // "found me"
    ```
  </Step>

  <Step title="Apply a patch with JsonPatch.Net">
    Install the package:

    ```bash
    dotnet add package JsonPatch.Net
    ```

    Deserialize a patch document and apply it:

    ```csharp
    using Json.Patch;
    using System.Text.Json;
    using System.Text.Json.Nodes;

    var patchJson = "[{\"op\":\"replace\",\"path\":\"/foo\",\"value\":\"world\"}]";
    var patch = JsonSerializer.Deserialize<JsonPatch>(patchJson);

    var doc = JsonNode.Parse("{\"foo\":\"hello\"}");
    var result = patch!.Apply(doc);

    Console.WriteLine(result.IsSuccess);  // true
    Console.WriteLine(result.Result);     // {"foo":"world"}
    ```

    You can also build patches inline:

    ```csharp
    var patch = new JsonPatch(
        PatchOperation.Add("/baz", "qux"),
        PatchOperation.Replace("/foo", "bar")
    );
    ```
  </Step>
</Steps>
